# 📊 Sales Analysis Dashboard (Excel)

## 📌 Overview

This project presents an interactive **Sales Analysis Dashboard** built using Microsoft Excel.
The dashboard focuses on analyzing **customer behavior, regional performance, and sales trends** to generate actionable business insights.

---

## 🎯 Objectives

* Analyze total sales, profit, and customer distribution
* Identify top-performing regions, cities, and segments
* Track monthly sales trends
* Provide business recommendations based on data

---

## 🛠️ Tools & Techniques

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers & Filters
* Data Cleaning & Transformation
* Dashboard Design & Data Visualization

---

## 📊 Dashboard Features

* **KPIs:** Total Customers, Cities, Segments
* **Regional Analysis:** Sales, Profit, Customers distribution
* **Customer Segmentation:** Consumer, Corporate, Home Office
* **Top Cities Analysis:** Orders & Customers
* **Time Series Analysis:** Monthly customer trends
* **Geo Visualization:** Customers by State (Map Chart)
* **Interactive Filters:** Year, Quarter, Category

---

## 🔍 Key Insights

* The **West region** leads in total sales
* The **Consumer segment** contributes the highest share of sales and orders (~50%+)
* Sales show a strong upward trend toward the end of the year, peaking in November
* Major cities like **New York** and **Los Angeles** dominate in both customers and orders

## 🚀 How to Use

1. Download the Excel file
2. Open using Microsoft Excel (recommended version: 2016 or later)
3. Use slicers to interact with the dashboard
4. Explore insights across regions, segments, and time
